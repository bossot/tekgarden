# TekGarden 🌱

**Reproducible homelab infrastructure** — Templates, documentation, and anonymized examples from our homelab setup.

This repository contains reusable templates and architecture documentation from TekGarden, our production homelab in Mallorca. All content is **anonymized** to be shared without exposing sensitive data.

## Contents

- **`boilerplates/`** — Reusable templates (Docker Compose, FluxCD, Ansible, OpenTofu)
- **`docs/`** — Architecture documentation, decisions, and runbooks (in Catalan)
- **`docs/en/`** — English version of the documentation

## Why TekGarden?

TekGarden was born from the need for a **reliable, reproducible, and documented** homelab using modern tools: Kubernetes, GitOps with FluxCD, infrastructure as code with OpenTofu, and configuration management with Ansible.
