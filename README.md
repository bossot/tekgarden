# TekGarden 🌱

**Infraestructura homelab reproduïble** — Plantilles, documentació i exemples anonimitzats de la configuració del nostre homelab.

Aquest repositori conté plantilles reutilitzables i documentació d'arquitectura del TekGarden, el nostre homelab de producció a Mallorca. Tot el contingut està **anonimitzat** per poder-lo compartir sense exposar dades sensibles.

## Contingut

- **`boilerplates/`** — Plantilles reutilitzables (Docker Compose, FluxCD, Ansible, OpenTofu)
- **`docs/`** — Documentació d'arquitectura, decisions i runbooks (en català)
- **`docs/en/`** — English version of the documentation

## Per què TekGarden?

TekGarden neix de la necessitat de tenir un homelab **fiable, reproduïble i documentat** amb eines modernes: Kubernetes, GitOps amb FluxCD, infraestructura com a codi amb OpenTofu, i gestió de configuracions amb Ansible.
