# Plantilles Docker Compose

Exemples de desplegaments Docker Compose anonimitzats.

## Contingut (properament)

- Servei web amb reverse proxy
- Base de dades amb backups automàtics
- Stack de monitoratge (Prometheus + Grafana)

## Estàndards

- Noms de servei descriptius
- Xarxes separades per propòsit
- Variables d'entorn via fitxer `.env`
- Volums amb drivers locals
