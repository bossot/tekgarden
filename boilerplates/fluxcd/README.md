# Plantilles FluxCD

Components GitOps reutilitzables per Kubernetes amb FluxCD.

## Contingut (properament)

- HelmRelease amb SourceRef
- Kustomization per entorns
- ExternalSecret amb 1Password
- Health check probes

## Estàndards

- Namespace aïllat per aplicació
- Labels estàndard de FluxCD
- Dependencies entre releases
- Alertes de reconcilació
