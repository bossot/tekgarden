# Mòduls OpenTofu

Infraestructura com a codi per provisionament de recursos.

## Contingut (properament)

- Mòdul per LXC a Proxmox
- Mòdul per xarxes i VLANs
- Gestió de DNS
- Configuració de firewall

## Estàndards

- Outputs sempre definits
- Variables amb descripció
- Backend remot (S3)
- Terraform docs generat automàticament
